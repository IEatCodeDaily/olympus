ADR 0018 · accepted architecture · prerequisite-gated
Olympus Observability Dataflow
OpenTelemetry-shaped diagnostics, permanent product truth, and bounded retention remain deliberately separate.

Permanent truth

Transcripts, lifecycle, permissions, capability decisions, and terminal state remain in the append-only Olympus journal. Debugging never becomes the only copy of state.

Provable diagnostics

Each attempt is a bounded trace. Authenticated source sequences and positive coverage distinguish complete, incomplete, unknown, and expired evidence.

Framework-neutral seam

Rust tracing emits safely; OpenTelemetry supplies typed causality and optional export. Hall derives tenancy—no LangSmith or Collector is required.
