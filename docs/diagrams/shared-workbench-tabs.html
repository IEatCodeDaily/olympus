Shared view semantics, surface-specific pane systems
Vault and Sessions share split-tree mechanics with different chrome. Cockpit stays deliberately single-pane. Domain truth remains outside presentation state.
Identity

Geometry
Vault and Sessions use recursive horizontal/vertical split trees; Cockpit stays flat.
Chrome
